# Age Calculator 

My first Python project built using Python.

## Features
- Takes the user's name
- Takes the birth year
- Takes the current year
- Calculates the user's age
- Displays the result

## Technologies Used
- Python

## How to Run
1. Open the project in VS Code.
2. Run `Age_Calculator.py`.
3. Enter your name, birth year, and current year.
4. The program will display your age.
